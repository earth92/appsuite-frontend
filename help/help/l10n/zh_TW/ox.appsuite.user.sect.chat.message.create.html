使用訊息

10.3.1. 閱讀、發送、回覆訊息

有下列選項：

  • 搜尋並閱讀聊天訊息。您會如何收到來訊通知：

    • 若有未讀的聊天訊息，選單列上的聊天圖示將以紅點標示。

    • 您可在聊天設定當中設定下列通知：

      • 在收到新訊息時是否要播放音效。

      • 若 Groupware 在背景執行，是否要顯示桌面通知。

      • 若您已離線，將會收到包含聊天訊息內容的電子郵件通知。若您回覆此郵件，回覆內容將插入到聊天訊息中。

  • 傳送訊息與檔案給群組或單一聯絡人

  • 回覆先前的訊息。將在您的回覆中引用原始訊息。

如何閱讀訊息：

  1. 從聊天清單選擇聊天。

    包含未讀訊息的聊天將以下列方式顯示：

    • 聊天名稱將以粗體顯示。

    • 包含未讀訊息數量的圖示。

  2. 讀取對話訊息。要檢視對話中先前的訊息，使用右方的圖示向上捲動。

選項：

  • 要搜尋聊天，在搜尋或開始新聊天輸入欄位輸入搜尋詞彙。

如何傳送訊息：

  1. 使用以下方式之一：

    • 要開始新聊天，點擊新增圖示。

      在選單中選擇會話類型。

    • 要使用現有聊天，從清單中選擇聊天。將顯示至今為止的聊天訊息。

  2. 在輸入欄位輸入訊息。

    若您在輸入訊息時關閉聊天視窗，或開啟另一個新視窗，輸入的內容將保留至登出 Groupware 為止。

  3. 要傳送訊息，按下 [Enter] 或點擊傳送 圖示 。

    若為私人聊天，訊息旁的圖示將包含訊息的讀取狀態：

    • 一個灰色的勾選符號：訊息已傳送到伺服器。

    • 兩個灰色的勾選符號：訊息已從伺服器傳送給收訊者。

    • 兩個藍色的勾選符號：收訊者已閱讀訊息。

選項：

  • 要在訊息中換行，按下 [Shift] + [Enter]。

  • 若您傳送訊息到群組，可以特別指定聯絡人。要這樣做，輸入 [@] 並接者要指定的人名與訊息。指定的聯絡人將會在清單中的群組名稱旁看到紅色標示。

  • 要傳送表情符號，點擊輸入欄位旁的加入表情符號圖示。將開啟包含表情符號的視窗。選擇至少一個表情符號。

    若要關閉視窗，再次點擊加入表情符號圖示。

  • 要傳送檔案，點擊輸入欄位旁的上傳檔案圖示。

    您也可以從檔案瀏覽器或桌面拖曳一個或多個檔案到聊天欄位。

    註：由於安全性因素，無法傳送可執行檔。

  • 要檢視所有先前收發過的檔案，點擊清單中的所有檔案。

  • 使用下列符號來格式化訊息文字：

    • 斜體：_文字內容_。

    • 粗體：*文字內容*。

    • 刪除縣：~文字內容~。

    • 單行程式碼：`文字內容`。

    • 多行程式碼：```文字內容

      第一行

      第二航

      ```

      註：若需換行程式碼，請按 [Shift]+[Enter]。

如何回覆訊息：

  1. 在聊天欄位，前往先前收到的訊息。

    點擊訊息旁的訊息操作圖示。選擇回覆。

    訊息與寄件者將在訊息欄位被引用。

  2. 在訊息欄位當中輸入您的回覆。

    要傳送回覆，按下 [Enter] 或點擊傳送 圖示 。

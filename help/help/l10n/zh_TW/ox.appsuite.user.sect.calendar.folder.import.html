7.8.3. 自檔案匯入行事曆

您可以匯入於其他應用程式中建立的行事曆。您想要匯入的檔案必須為 iCal 格式。

如何從檔案匯入行事曆：

  1. 點擊資料夾畫面中的新增行事曆。選單開啟。

    點選 上傳檔案。從檔案會入 視窗將打開。

  2. 點選 上傳檔案。選擇檔案。

  3. 輸入新行事曆的名稱。單擊 匯入。

結果：將建立新行事曆。

註：將移除所有匯入的約會參與者。將加入您作為參與者。

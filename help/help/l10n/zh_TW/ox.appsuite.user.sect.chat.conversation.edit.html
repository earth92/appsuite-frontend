使用聊天

10.2.2. 編輯聊天

所有參與者都可以編輯群組聊天與公開頻道。有下列選項：

  • 變更圖片

  • 變更名稱

  • 針對群組聊天：新增或刪除參與者

如何編輯群組聊天或公開頻道：

  1. 從聊天清單中選擇群組聊天或公開頻道。

  2. 點選右上角的更多操作圖示。

    選擇編輯聊天或編輯頻道。將開啟編輯視窗。

  3. 您可以更改下列項目：

    • 點擊圖片來更改圖片

    • 變更名稱

    • 針對群組聊天：新增或刪除參與者

  4. 點擊儲存。

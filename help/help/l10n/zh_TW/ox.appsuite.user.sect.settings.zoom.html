基礎設定

3.7. 建立並設定 Zoom 整合

有下列選項：

  • 設定 Zoom 整合

    在能與其他聯絡人進行 Zoom 會議前，必須先設定好 Zoom 整合。為此，您需要授權讓 Groupware 存取您的 Zoom 帳號。

  • 設定 Zoom 整合

    您可以按需求設定 Zoom 整合。

如何設定 Zoom 整合：

  1. 若您在 電子郵件、通訊錄 or 行事曆 應用程式尚未設定 Zoom 整合前，使用 Zoom 撥打給聯絡人，將顯示註冊 Zoom 帳號的視窗。這種情況下，請跳到第 4 步。

  2. 點擊選單列右邊的設定圖示 。點選設定。

    點選側邊欄的 Zoom 整合。

  3. 點選顯示區域中的 新增 Zoom 帳號。若未顯示此按鈕，表示 Zoom 整合已經完成。

    將開啟用來註冊 Zoom 帳號的視窗。

  4. 使用以下方式之一：

    • 在登入下方，輸入管理員或主機商提供給您的登入資訊。

    • 若要註冊自己的 Zoom 帳號，點擊免費註冊。按照說明操作。

    將顯示連結的 Zoom 帳號資料。

    要取消連結，點選帳號旁的移除帳號圖示 。

如何設定 Zoom 整合：

  1. 點擊選單列右邊的設定圖示 。點選設定。

  2. 點選側邊欄的 Zoom 整合。

有以下選項：

約會

透過這些選項，您可以指定在建立約會時，點擊會議下拉選單當中的Zoom 會議時的行為。

  • 總是加入隨機會議密碼. 定義是否要使用自動產生的密碼加密會議。

  • 自動複製鏈結到會議位置. 設定是否要自動將 Zoom 會議參加連結複製到位置欄位。

  • 將撥入號碼複製到描述. 設定是否要自動將 Zoom 會議參加連結複製到位置欄位。

來電

透過這些選項，您可以指定當 Groupware 在背景收到 Zoom 來電時的行為。

  • 顯示桌面通知. 設定是否要在收到 Zoom 來電時顯示桌面通知。

  • 收到來電時播放鈴聲. 指定在收到新 Zoom 來電時是否要播放音效。

撥入號碼

若您有 Zoom Pro 以上等級的帳號，Zoom 將提供各國的撥入號碼。要設定在約會邀請中顯示的國家，點擊開啟 Zoom 設定檔。

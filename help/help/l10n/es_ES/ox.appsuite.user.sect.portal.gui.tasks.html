4.1.6. Widget Tareas

Muestra las tareas sin terminar. Puede hacer lo siguiente:

  • Si se pulsa en una tarea, se abrirá un elemento emergente. El elemento emergente muestra los datos de la cita.

  • Para editar la tarea, pulse en el botón correspondiente del elemento emergente.

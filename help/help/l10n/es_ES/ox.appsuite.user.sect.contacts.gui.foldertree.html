Los componentes de Libreta de direcciones

6.1.2. La vista de carpeta de Libreta de direcciones

Muestra las libretas de direcciones.

Para mostrar la vista de carpeta, pulse en Vista en la barra de herramientas. Marque Vista de carpeta.

Contenido

  • Mis libretas de direcciones. Contiene sus libretas de direcciones personales.

  • Añadir nueva libreta de direcciones. Abre un menú con funciones para crear libretas de direcciones y para suscribirse a libretas de direcciones externas o compartidas.

  • Mis datos de contacto. Abre una ventana en la que se pueden configurar sus datos de contacto en la libreta de direcciones global.

  • Libretas de direcciones públicas. Contiene libretas de direcciones compartidas con todos los usuarios.

  • Libretas de direcciones compartidas. Contiene libretas de direcciones compartidas con usted por otros usuarios.

Dependiendo de la configuración, las libretas de direcciones públicas y las libretas de direcciones compartidas pueden no estar disponibles.

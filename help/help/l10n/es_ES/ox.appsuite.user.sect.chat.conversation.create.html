10.2.1. Creación de conversaciones

Existen las siguientes opciones:

  • crear una conversación privada con un contacto individual

  • crear una conversación grupal con participantes concretos

  • crear un canal público al que otros contactos puedan unirse

Cómo crear una conversación:

  1. Pulse el icono Nueva que hay encima de la lista de conversaciones, o en Nueva conversación.

    Se abre un menú.

  2. Use uno de los siguientes métodos:

    • Para crear una conversación con un solo contacto, seleccione Conversación privada. Se abre la ventana Iniciar nueva conversación.

      Seleccione un contacto.

      Pulse en Iniciar conversación.

    • Para crear una conversación con varios participantes, seleccione Conversación grupal. Se abre la ventana Crear conversación grupal.

      Indique un nombre de grupo. Puede añadir una foto pulsando en el icono.

      Añadir contactos.

      Pulse en Crear conversación.

    • Para crear un canal público al que otros contactos puedan unirse, seleccione Canal. Se abre la ventana Crear nuevo canal.

      Indique un nombre de canal. Puede añadir una foto pulsando en el icono.

      Pulse Crear canal.

La nueva conversación se mostrará en la lista.

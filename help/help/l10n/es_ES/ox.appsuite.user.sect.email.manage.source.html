5.7.6. Mostrar código fuente del correo

El código fuente del correo contiene el contenido completo de un correo, es decir, los datos de la cabecera del correo completos.

Cómo mostrar el código fuente de un correo:

  1. Seleccione un correo.

  2. Pulse el icono Más acciones de la barra de herramientas o en la vista de detalle. Seleccione Ver código fuente.

    También puede usar el menú contextual de la lista de correos.

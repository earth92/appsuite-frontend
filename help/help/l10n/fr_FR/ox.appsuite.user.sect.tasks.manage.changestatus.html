8.9.4. Modifier l'état de confirmation d'une tâche

Vous pouvez ultérieurement l'état de confirmation d'une tâche.

Pour modifier la confirmation d'une tâche :

  1. Choisissez une tâche dans la liste.

  2. Cliquez sur l'icône Actions supplémentaires dans la barre d'outils. Cliquez sur Modifier l'état de confirmation.

  3. Dans la fenêtre qui s'ouvre, choisissez l'état de confirmation désiré. Si nécessaire, saisissez un message.

  4. Vous pouvez saisir un commentaire dans la fenêtre Modifier l'état de confirmation. Cliquez sur l'un des boutons Accepter, Provisoire ou Décliner.

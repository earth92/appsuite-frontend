5.7.6. Afficher le code source d'un message

Le code source d'un message électronique représente son contenu complet, y compris toutes les données de ses en-têtes.

Pour afficher le code source du message :

  1. Sélectionnez un courriel.

  2. Cliquez sur l'icône Actions supplémentaires dans la barre d'outils ou dans la vue détaillée. Choisissez Voir le code source.

    Vous pouvez également utiliser le menu contextuel dans la vue en liste des courriels.

10.3.2. Gérer les messages

Vous disposez des options suivantes :

  • modifier ses propres messages

  • supprimer ses propres messages

Pour modifier un message que vous avez envoyé :

  1. Dans le champ de discussion instantanée, accédez à un message que vous avez envoyé.

  2. Cliquez sur l'icône Actions sur les messages à côté du message. Sélectionnez Modifier le message.

  3. Modifiez le message.

    Pour conclure et envoyer le message, appuyez sur la touche [Entrée] ou cliquez sur l'icône Envoyer .

    Pour annuler la modification, cliquez sur l'icône Annuler .

Conseil : pour modifier le message envoyé précédemment, appuyez sur la flèche Haut.

Pour supprimer un message que vous avez envoyé :

  1. Dans le champ de discussion instantanée, accédez à un message que vous avez envoyé.

  2. Cliquez sur l'icône Actions sur les messages à côté du message. Sélectionnez Supprimer le message.

Interface utilisateur :

  • La fenêtre Discussion

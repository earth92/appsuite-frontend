Chapitre 4.  Portail

Apprenez à utiliser l'application Portail comme source centrale d'informations concernant vos courriels entrants, vos rendez-vous et tâches programmés, et vous actualités en provenance des réseaux sociaux.

Pour démarrer l'application Portail :

  • Cliquez sur l'icône Toutes les applications dans la barre de menu. Dans le lanceur d'applications, cliquez sur Portail.

9.4.1. Créer des dossiers Fichiers

Vous pouvez créer de nouveaux dossiers dans le dossier actuel.

Pour créer un nouveau dossier :

  1. Dans la vue des dossiers, ouvrez un dossier Fichiers dans lequel vous avez le droit de créer des objetss.

  2. Cliquez sur Nouveau dans la barre d'outils. Cliquez sur Dossier.

    Dans la vue des dossiers, vous pouvez également cliquer sur l'icône Actions à côté du nom du dossier. Cliquez sur Ajouter un nouveau dossier.

  3. Saisissez un nom dans la fenêtre Ajouter un nouveau dossier. Cliquez sur Ajouter.

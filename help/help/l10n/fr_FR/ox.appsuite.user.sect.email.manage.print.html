5.7.12. Imprimer des courriels

Vous pouvez imprimer le contenu d'un ou plusieurs courriels.

Pour imprimer le contenu d'un courriel :

  1. Sélectionnez un ou plusieurs courriels.

  2. Cliquez sur l'icône Actions supplémentaires dans la barre d'outils ou dans la vue détaillée. Dans le menu, choisissez Imprimer.

    Vous pouvez également utiliser le menu contextuel dans la vue en liste des courriels.

    Une fenêtre d'aperçu avant impression s'ouvre alors.

  3. Si nécessaire, changez les réglages d'impression. Cliquez sur le bouton Imprimer.

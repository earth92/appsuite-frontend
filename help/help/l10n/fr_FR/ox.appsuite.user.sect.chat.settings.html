10.5. Réglages des discussions instantanées

Pour utiliser les réglages des discussions instantanées :

  1. Cliquez sur l'icône Réglages à droite de la barre de menu. Cliquez sur Réglages.

  2. Dans la liste, cliquez sur Discussion instantanée.

Suivant la configuration, les réglages suivants sont disponibles :

Options d'affichage

  • sélectionner la dernière discussion au démarrage. Définit si la dernière discussion utilisée doit être affichée dans la fenêtre de discussion une fois que vous êtes connecté au collecticiel.

  • Regrouper les discussions instantanées par type. Définit si les discussions doivent être regroupées en discussions privées, discussions de groupe et salons.

  • Trier les discussions instantanées par. Définit l'ordre des discussions dans la liste.

  • densité de la liste. Définit le niveau de détail pour l'affichage des discussions dans la liste.

Notifications

  • afficher les notifications de bureau pour les nouveaux messages. Définit si vous devez recevoir une notification de bureau pour les messages entrants lorsque le collecticiel est en arrière-plan. Ce réglage ne prend effet que si vous avez également activé les notifications de bureau dans les réglages de base (voir Gérer les notifications).

  • jouer un son de notification pour les nouveaux messages. Définit si un son est joué à l'arrivée d'un message. Si les sons de notification sont activés, des réglages supplémentaires sont disponibles :

    • le son à jouer

    • les circonstances dans lesquelles jouer ce son

  • notifications par courriel lorsque vous êtes déconnecté. Définit si vous recevez des notifications par courriel, et pour quelles discussions, lorsque vous êtes déconnecté.

10.2.1. Créer des discussions instantanées

Vous disposez des options suivantes :

  • créer une discussion privée avec un contact unique

  • créer une discussion de groupe avec des participants donnés

  • créer un salon public que d'autres contacts peuvent rejoindre

Pour créer une discussion instantanée :

  1. Cliquez sur l'icône Nouveau au-dessus de la liste des discussions ou sur Nouvelle discussion.

    Un menu s'ouvre alors.

  2. Utilisez l'une des méthodes suivantes :

    • Pour créer une discussion avec un contact unique, choisissezDiscussion privée. La fenêtre Démarrer une nouvelle conversation s'ouvre alors.

      Sélectionnez un contact.

      Cliquez sur Démarrer une conversation.

    • Pour créer une discussion comportant plusieurs participants, sélectionnez Discussion de groupe. La fenêtre Créer une discussion de groupe s'ouvre alors.

      Saisissez un nom pour le groupe. Vous pouvez ajouter une image en cliquant sur l'icône.

      Ajoutez des contacts.

      Cliquez sur Créer une discussion.

    • Pour créer un salon public que d'autres contacts peuvent rejoindre, sélectionnez Salon. La fenêtre Créer un nouveau salon s'ouvre alors.

      Saisissez un nom pour le salon. Vous pouvez ajouter une image en cliquant sur l'icône.

      Cliquez sur Créer un salon.

La nouvelle discussion apparaîtra dans la liste.

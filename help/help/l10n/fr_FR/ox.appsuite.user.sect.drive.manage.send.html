9.7.1. Envoyer des fichiers comme pièces jointes

Vous pouvez envoyer les versions courantes des fichiers comme pièces jointes à un courriel.

Pour envoyer des fichiers comme pièces jointes à un courriel :

  1. Sélectionnez un ou plusieurs fichiers dans la zone d'affichage.

  2. Utilisez l'une des méthodes suivantes :

    • Cliquez sur l'icône Actions supplémentaires dans la barre d'outils. Cliquez sur Envoyer par courriel.

    • Cliquez sur Envoyer par courriel dans le menu contextuel.

    • Utilisez l'icône Actions supplémentaires dans l'Afficheur.

  3. Dans la fenêtre de rédaction d'un courriel, complétez les détails pour envoyer un nouveau courriel.

Interface utilisateur :

4.2.4. Ajout d'un composant de portail pour les réseaux sociaux

Pour accéder aux informations et fonctions de vos réseaux sociaux, vous pouvez ajouter des composants de réseaux sociaux.

Pour ajouter un composant permettant l'accès aux réseaux sociaux :

  1. Cliquez sur Ajouter un composant dans le coin supérieur droit de l'application Portail. Sélectionnez l'un des éléments. Le composant est ajouté.

  2. Pour activer l'accès, ajoutez votre compte de réseau social en cliquant sur le bouton respectif dans le composant.

    Vous pouvez modifier ou supprimer le compte dans les réglages des comptes.

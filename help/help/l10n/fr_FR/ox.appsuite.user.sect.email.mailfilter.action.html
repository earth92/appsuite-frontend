5.8.3. Ajouter une action

Utilisez une action pour définir ce qu'il advient d'un courriel lorsqu'il correspond aux conditions. Vous pouvez définir une ou plusieurs actions.

Pour ajouter une action depuis la fenêtre Créer une nouvelle règle :

  1. Cliquez sur Ajouter une action. Choisissez une action dans le menu.

  2. Suivant l'action choisie, des boutons et champs de saisie supplémentaires peuvent s'afficher. Utiliser ces outils pour renseigner les détails de l'action.

  3. Vous pouvez ajouter d'autres actions.

    Pour supprimer une action, cliquez sur l'icône Supprimer à côté de l'action.

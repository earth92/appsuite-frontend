4.2.3. Ajout de composants au portail

Vous pouvez compléter les composants existants en en ajoutant de nouveaux.

Pour ajouter de nouveaux composants :

  1. Utilisez l'une des méthodes suivantes :

    • Cliquez sur l'icône Réglages à droite de la barre de menu. Cliquez sur Réglages.

      Cliquez sur Portail dans la barre latérale. Cliquez sur Ajouter un composant graphique. Choisissez un élément.

    • Cliquer sur Ajouter un composant dans le coin supérieur droit de l'application Portail. Sélectionnez un élément.

  2. Certains composants ont besoin d'informations supplémentaires. Saisissez les valeurs demandées.

Interface utilisateur :

  • Bouton « Ajouter un composant »

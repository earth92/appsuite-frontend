Chapitre 6.  Carnet d'adresses

Découvrez comment enregistrer, utiliser et organiser vos contacts dans l'application Carnet d'adresses.

Pour démarrer l'application Carnet d'adresses :

  • Cliquez sur l'icône Toutes les applications dans la barre de menu. Dans le lanceur d'applications, cliquez sur Carnet d'adresses.

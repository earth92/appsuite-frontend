Chapitre 9.  Fichiers

Découvrez comment utiliser l'application Fichiers pour enregistrer et afficher des fichiers, documents et autres médias, et les utiliser de manière collaborative avec d'autres personnes.

Pour démarrer l'application Fichiers :

  • Cliquez sur l'icône Toutes les applications dans la barre de menu. Dans le lanceur d'applications, cliquez sur Fichiers.

    Suivant la configuration, vous pouvez également cliquer sur l'icône de démarrage rapide de la barre de menu pour lancer l'application Fichiers.

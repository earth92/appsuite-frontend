7.4.6. Utiliser les rappels de rendez-vous

Lorsque vous créez ou modifiez un rendez-vous, vous pouvez demander à recevoir un rappel de la date d'échéance du rendez-vous. Pour cela vous pouvez faire appel à l'une ou plusieurs des notifications suivantes :

  • un rappel de rendez-vous dans la zone de notification

  • un rappel de rendez-vous par courriel

Pour définir un rappel de rendez-vous dans la fenêtre d'édition de rendez-vous :

  1. Cliquez sur le réglage courant sous Rappel. La fenêtre Modifier les rappels s'ouvre. Elle affiche le rappel actuellement configuré.

  2. Pour modifier le rappel actuellement configuré, sélectionnez le type de rappel. Choisissez une heure pour le rappel. Définissez la référence horaire.

  3. Pour ajouter un autre rappel, cliquez sur Ajouter un nouveau rappel.

    Pour supprimer un rappel, cliquez sur l'icône Supprimer .

  4. Cliquez sur Appliquer.

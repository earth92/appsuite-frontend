10.2.3. Rejoindre une discussion instantanée

Pour rejoindre des discussions de groupe ou des salons publics, vous disposez des options suivantes :

  • Une discussion de groupe n'est visible que par les membres du groupe. Vous ne pouvez pas rejoindre de vous-même une discussion de groupe.

  • En revanche, les salons publics sont visibles par l'ensemble des utilisateurs. Vous pouvez rejoindre un salon public à tout moment.

Pour rejoindre une discussion de groupe :

  • Demandez à un membre du groupe de vous ajouter à la discussion.

Pour rejoindre un salon public :

  1. En bas de la liste des discussions, cliquez sur Tous les salons. Les salons publics sont alors affichés dans le champ de discussion instantanée.

  2. Utilisez l'une des méthodes suivantes :

    • Pour rejoindre directement un salon, cliquez sur Rejoindre à côté de ce salon.

    • Pour consulter les messages passés ainsi que les membres d'un salon, cliquez sur celui-ci.

      Pour rejoindre le salon, cliquez sur Rejoindre sous la fenêtre de discussion instantanée.

Interface utilisateur :

  • La fenêtre Discussion

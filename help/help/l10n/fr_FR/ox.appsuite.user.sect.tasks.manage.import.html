8.9.5. Importer des tâches

Vous pouvez importer des tâches depuis des fichiers au format iCal.

Pour importer des tâches depuis un fichier iCal :

  1. Dans la vue des dossiers, sélectionnez le dossier vers lequel importer les tâches.

  2. Cliquez sur l'icône Actions à côté du dossier de tâches. Cliquez sur Importer.

  3. Dans la fenêtre Importer depuis un fichier, cliquez sur Transférer un fichier. Choisissez un fichier au format iCal.

    Pour importer également des tâches ayant le même identifiant que des tâches existantes, activez l'optionIgnorer les événements existants.

  4. Cliquez sur Importer.

Résultat : les tâches sont ajoutées au dossier.

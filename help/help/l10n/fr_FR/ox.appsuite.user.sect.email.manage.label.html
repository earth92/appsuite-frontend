5.7.5. Classement des courriels

Suivant la configuration du collecticiel, différentes options sont disponibles pour classer les courriels. Vous disposez des options suivantes :

  • des étiquettes de couleur

  • un drapeau

Pour classer un courriel :

  1. Sélectionnez un ou plusieurs courriels.

  2. Pour ajouter une étiquette de couleur, cliquez sur l'icône Définir la couleur dans la barre d'outils ou dans la vue détaillée. Choisissez la couleur.

    Pour supprimer une étiquette, cliquez à nouveau sur l'icône dans la barre d'outils ou dans la vue détaillée. Cliquez sur Aucune.

  3. Pour ajouter un marquage, cliquez sur l'icône Marquer dans la barre d'outils ou dans la vue détaillée.

    Vous pouvez également utiliser le menu contextuel dans la vue en liste des courriels.

    Pour retirer un marquage, utilisez l'une des méthodes suivantes :

    • Cliquez à nouveau sur l'icône dans la barre d'outils ou dans la vue détaillée.

    • Utilisez le menu contextuel dans la vue en liste des courriels.

Conseil : Pour trier les courriels par marquage, cliquez sur Trier par au-dessus de la liste.

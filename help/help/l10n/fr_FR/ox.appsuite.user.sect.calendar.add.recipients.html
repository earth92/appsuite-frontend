7.4.12. Inviter les destinataires d'un courriel à un nouveau rendez-vous

Si un courriel comporte plusieurs destinataires, vous pouvez tous les inviter à un nouveau rendez-vous.

Pour inviter tous les destinataires d'un courriel à un nouveau rendez-vous :

  1. Sélectionnez un courriel dans l'application Courriel.

  2. Cliquez sur l'icône Actions supplémentaires dans la vue détaillée.

    Cliquez sur Inviter à un rendez-vous.

  3. Renseignez les détails requis pour créer le rendez-vous.

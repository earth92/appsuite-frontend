Discussion

10.1. La fenêtre Discussion

Cette fenêtre est utilisées pour toute les fonctions de Discussion :

  • recherche et affichage de contacts, de groupes et de salons publics

  • création et modification de groupes et de salons publics

  • échange de messages directs avec un contact individuel, échange de messages au sein d'un groupe ou d'un salon public

Pour ouvrir la fenêtre de discussion, effectuez l'une des actions suivantes :

  • Cliquez sur l'icône Discussion instantanée dans la barre de menu.

    Si vous avez des messages non lus, l'icône présente un point rouge.

  • Utilisez le lanceur d'applications dans la barre de menu.

Soit la fenêtre de discussion instantanée est ancrée à droite, soit vous pouvez la déplacer. La barre de titre contient des icônes permettant de basculer la vue.

  • Si la fenêtre de discussion est ancrée : pour pouvoir la déplacer librement, cliquez sur l'icône Détacher la fenêtre .

  • Si la fenêtre de discussion peut être déplacée : pour l'ancrer, cliquez sur l'icône Ancrer à droite .

    Pour ajuster la largeur, faites glisser vers la gauche ou vers la droite la bordure séparant la fenêtre de la vue détaillée.

La liste de discussions et le champ de discussion constituent des éléments essentiels de la fenêtre de discussion. Le champ de discussion affiche les messages de la discussion sélectionnée dans la liste.

  • La fenêtre de discussion ancrée affiche soit la liste de discussions, soit le champ de discussion.

    • Si le champ de discussion est affiché, vous pouvez basculer vers la liste des discussions en cliquant sur Discussions instantanées sur la gauche de la barre de titre.

    • Si la liste de discussions est affichée, vous pouvez basculer vers le champ de discussion en cliquant sur une discussion. La barre de titre du champ de discussion affiche le nom de la discussion sélectionnée dans la liste.

  • La fenêtre de discussion déplaçable affiche la liste des discussions et le champ de discussion côte à côte.

Contenu

  • La fenêtre de discussion déplaçable affiche : votre image de profil, votre nom d'affichage, votre état de présence

  • fonctions permettant de créer de nouvelles discussions instantanées :

    • dans la fenêtre de discussion déplaçable : l'icône Nouveau

    • dans la fenêtre de discussion ancrée : Nouvelle discussion

    Un clic dessus ouvre un menu comportant plusieurs fonctions :

    • créer une discussion privée avec un contact unique

    • créer une discussion de groupe avec plusieurs participants

    • créer un salon public que d'autres contacts peuvent rejoindre

  • au-dessus de la liste des discussions : champ de saisie Rechercher ou démarrer une discussion instantanée. La saisie d'un terme à rechercher fait apparaître les éléments suivants dans la liste des discussions :

    • Contacts, groupes et salons publics dont le nom correspond au terme recherché

      Un clic sur un contact démarre une discussion privée avec ce contact

    • Discussions contenant le terme recherché

  • Discussions instantanées. Contient la liste des discussions ouvertes. Via les réglages des discussions, vous pouvez définir si les discussions doivent être regroupées en Discussions privées, Discussions de groupe et Salons.

    • Pour les discussions privées, l'état de présence du contact est affiché.

    • Lorsque vous cliquez sur une discussion instantanée, les messages envoyés et reçus dans le cadre de cette discussion s'affichent.

    • La vue en liste des discussions peut être ajustée dans les réglages des discussions instantanées.

  • En dessous de la liste des discussions figurent des fonctions permettant d'afficher des éléments spécifiques dans le champ de discussion :

    • Tous les salons. Affiche dans le champ de discussion tous les salons publics. Vous pouvez rejoindre un salon en cliquant sur Rejoindre.

    • Tous les fichiers. Affiche dans le champ de discussion une liste des fichiers utilisés dans les discussions.

    • Historique. Affiche dans le champ de discussion toutes les discussions ouvertes et fermées.

    Si la fenêtre de discussion est ancrée, le champ de discussion sera affiché automatiquement si vous cliquez sur l'une de ces fonctions.

  • Le titre du champ de discussion instantanée

    • Si la fenêtre de discussion est ancrée : Discussions instantanées. Bascule vers la liste des discussions.

    • Nom du contact, du groupe ou du salon sélectionné dans la liste des discussions.

      Pour les discussions privées, l'état de présence du contact est affiché.

      Pour les groupes ou les salons, les noms des membres sont également affichés.

      Un clic sur l'icône Actions supplémentaires ouvre un menu proposant des fonctions supplémentaires :

      • Fermer la discussion instantanée. Ferme la discussion instantanée actuelle et les messages associés. Pour afficher de nouveau cette discussion, cliquez sur Historique dans la liste.

      • Pour les discussions de groupe ou pour un salon :

        Modifier la discussion ouvre une fenêtre permettant de modifier la discussion.

        Vous pouvez quitter la discussion en cliquant sur Quitter la discussion instantanée.

    • Affiche le nom de la fonction que vous avez sélectionnée sous le champ de discussion instantanée.

  • Champ de discussion instantanée

    • Contient les messages envoyés et reçus dans le cadre de la discussion sélectionnée dans la liste.

      Pour les discussions privées, une icône indique l'état de lecture du message :

      • Une coche grise : le message est arrivé sur le serveur.

      • Deux coches grises : le message a été transféré au destinataire par le serveur.

      • Deux coches bleues : le message a été lu par le destinataire.

    • Affiche les notifications système si des discussions de groupe ou des salons publics sont créés ou modifiés.

    • Affiche le résultat de la fonction que vous avez sélectionnée sous le champ de discussion instantanée.

  • Éléments permettant de rédiger un message

    • Icône Transférer un fichier . Ajoute des fichiers à la discussion.

    • Champ de message. Celui-ci vous permet de saisir un message à envoyer dans le cadre de cette discussion instantanée.

    • Icône Ajouter un émoji . Ouvre ou ferme la fenêtre d'ajout d'émojis.

    • Icône Envoyer . Envoie le message.

5.7.11. Importer des courriels

Vous pouvez importer un message au format EML.

Pour importer un courriel :

  1. Ouvrez le dossier de courriels depuis lequel importer les messages.

  2. Faites glisser le fichier eml depuis un navigateur de fichiers ou depuis le bureau vers la liste des courriels.

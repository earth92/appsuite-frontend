5.7.10. Enregistrement des courriels

Vous pouvez enregistrer le courriel comme fichier texte. Le fichier texte possède l'extension EML. Si vous sélectionnez plusieurs courriels, un fichier EML sera créé pour chaque message. Ils seront ensuite compressés dans une archive ZIP.

Pour enregistrer un courriel :

  1. Sélectionnez un ou plusieurs courriels.

  2. Cliquez sur l'icône Actions supplémentaires dans la barre d'outils ou dans la vue détaillée. Choisissez Enregistrer en tant que fichier.

  3. Complétez les informations pour enregistrer le fichier.

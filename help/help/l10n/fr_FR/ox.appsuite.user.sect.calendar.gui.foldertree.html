7.1.3. La vue des dossiers de Agenda

Affiche les agendas.

Pour afficher la vue des dossiers, cliquez sur Vue dans la barre d'outils. Activez la Vue des dossiers.

Contenu

  • Mes agendas. Contient vos agendas personnels.

  • Ajouter un agenda. Ouvre un menu comportant des fonctions de création, d'abonnement ou d'importation d'agenda.

  • Agendas publics. Contient les agendas partagés avec tous les utilisateurs.

    Le dossier Tous mes rendez-vous publics contient l'ensemble des rendez-vous publics auxquels vous participez.

  • Agendas partagés. Contient les agendas partagés avec vous par d'autres utilisateurs.

    Si l'agenda partagé provient d'une autre installation du collecticiel, le nom d'utilisateur se verra complété du nom de l'autre domaine.

Suivant la configuration du collecticiel, le contenu peut différer des informations présentées ici.

Description générale de l'interface utilisateur :

  • La vue des dossiers

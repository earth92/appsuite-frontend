9.7.6. Enregistrement de fichiers comme PDF

Vous pouvez enregistrer des documents et fichiers texte au format PDF. Le fichier PDF sera enregistré dans le dossier contenant le document ou le fichier texte.

Pour enregistrer un fichier au format PDF :

  1. Sélectionnez un fichier dans la zone d'affichage.

  2. Utilisez l'une des méthodes suivantes :

    • Cliquez sur l'icône Actions supplémentaires dans la barre d'outils. Choisissez Enregistrer comme PDF.

    • Choisissez Enregistrer comme PDF depuis le menu contextuel.

    • Utilisez l'icône Actions supplémentaires dans l'Afficheur.

  3. Dans la fenêtre Enregistrer en PDF, vous pouvez saisir le nom du fichier PDF. Cliquez sur Enregistrer.

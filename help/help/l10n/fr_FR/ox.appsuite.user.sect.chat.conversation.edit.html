10.2.2. Modifier des discussions instantanées

Les discussions de groupe et salons publics peuvent être modifiées par l'ensemble des participants. Vous disposez des options suivantes :

  • modifier l'image

  • modifier le nom

  • Pour les discussions de groupe : ajouter ou retirer des participants

Pour modifier une discussion de groupe ou un salon public :

  1. Dans la liste des discussions, sélectionnez une discussion de groupe ou un salon public.

  2. Cliquez sur l'icône Actions supplémentaires en haut à droite.

    Sélectionnez Modifier la discussion ou Modifier le salon. Une fenêtre d'édition s'ouvre.

  3. Vous pouvez modifier les éléments suivants :

    • modifier l'image en cliquant dessus

    • modifier le nom

    • Pour les discussions de groupe : ajouter ou retirer des participants.

  4. Cliquez sur Enregistrer.

Interface utilisateur :

  • La fenêtre Discussion

7.9.5. Utiliser les couleurs de l'agenda

Vous pouvez assignez une couleur à un agenda. Tous les rendez-vous de cet agenda seront alors affichés dans la couleur de l'agenda. Si vous assignez une couleur individuelle à un rendez-vous lorsque vous le créez ou le modifiez, le rendez-vous sera affiché dans la couleur choisie et non celle de l'agenda.

Pour choisir une couleur d'agenda :

  1. Cliquez sur l'icône Actions à côté du nom de l'agenda dans la vue des dossiers.

  2. Choisissez une couleur dans la liste déroulante de choix des couleurs.

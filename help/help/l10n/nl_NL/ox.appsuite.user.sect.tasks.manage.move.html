8.9.3. Taken verplaatsen naar een andere map

U kan taken in één keer verplaatsen naar een andere map.

Het verplaatsen van taken naar een andere map gaat als volgt:

  1. Kies één of meerdere taken uit de lijst.

  2. Klikt Meer acties pictogram in de werkbalk. Klik op Verplaatsen.

  3. Selecteer een map met het Verplaatsen scherm. U kan een nieuwe map maken door op Map aanmaken te klikken.

  4. Klik op Verplaatsen.

Tip: U kan ook de geselecteerde objecten verplaatsen door deze uit het mappenoverzicht naar een andere map te slepen.

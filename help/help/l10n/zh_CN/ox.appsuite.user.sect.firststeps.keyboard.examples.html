键盘输入

2.5.3. 使用情况

显示如何使用键盘控制的指南：

  • 如何使用键盘发送一个新电子邮件

  • 如何使用键盘以回复一个电子邮件

如何使用键盘发送一个新电子邮件：

  1. 如果电子邮件应用程序并没有执行，请进行以下操作：

    • 在 Mac OS 系统上重复按下 [Ctrl]+[F6] 或 [F6] 直至 我的户口 图标在菜单条的右边被高亮化。

    • 使用 [Left arrow] 以导航到 所有应用程序 图标。请按下 [Enter] 以打开应用程序执行器。

    • 使用 [Down arrow] 以导航到 电子邮件 图标。按 [Enter]。

      电子邮件应用程序将会执行。

  2. 重复按下 [Tab] 直到撰写 按钮在工具条里高亮化。按 [Enter]。会打开撰写窗口。启用了输入字段To:。

  3. 输入收件人的电子邮件地址。按 [Enter]。

  4. 重复按下 [Tab] 直到主题输入字段为高亮。输入主题。按 [Enter]。会启用电子邮件文本输入字段。

  5. 输入电子邮件正文。

  6. 为了发送电子邮件，请按下 [Tab]。发送 按钮变为高亮。按 [Enter]。电子邮件被发送。

  7. 要添加附件，重复按下 [Tab] 或 [Shift]+[Tab] 直到附件 按钮为高亮。按 [Enter]。选择添加本地文件。

    会打开选择文件的对话字段。视系统而定，使用 [Tab] 键、[Enter] 键、方向键一个选择一个文件后关闭系统对话窗口。

    要添加附件，重复按下 [Tab] 直到发送按钮高亮。按 [Enter]。电子邮件被发送。

如何使用键盘以回复一个电子邮件

  1. 如果电子邮件应用程序并没有执行，请进行以下操作：

    • 在 Mac OS 系统上重复按下 [Ctrl]+[F6] 或 [F6] 直至 我的户口 图标在菜单条的右边被高亮化。

    • 使用 [Left arrow] 以导航到 所有应用程序 图标。请按下 [Enter] 以打开应用程序执行器。

    • 使用 [Down arrow] 以导航到 电子邮件 图标。按 [Enter]。

      电子邮件应用程序将会执行。

  2. 在 Mac OS 系统上重复按下 [Ctrl]+[F6] 或 [F6] 以在文件夹视图里一个文件夹高亮化。使用方向键选择需要的文件夹。要检视或隐藏子文件夹，请使用 [Right arrow] 或 [Left arrow]。

  3. 如果需要的文件夹以高亮，请按 [Enter]。文件夹里列表的第一个电子邮件会高亮化。

  4. 使用方向键选择需要的电子邮件。

  5. 要回复电子邮件，重复按下 [Tab] 或 [Shift]+[Tab]，直到回复 功能或回复全部功能为高亮。按下 [Enter] 以激活功能。

    重复按下 [Shift]+[Tab] 直到撰写 按钮在工具条里高亮化。使用方向键以浏览到 回复发送人 图标 或到 回复所有收件人 图标。按 [Enter]。

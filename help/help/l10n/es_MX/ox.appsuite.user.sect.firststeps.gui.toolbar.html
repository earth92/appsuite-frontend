2.4.4. La barra de herramientas

Nota: El contenido de la barra de herramientas es diferente para cada aplicación. La siguiente es una representación esquemática.

Contenido

  • Botones para crear objetos nuevos, por ejemplo un nuevo correo o un contacto nuevo.

  • Botones o iconos para editar objetos.

  • Dependiendo de la aplicación, podrían estar disponibles otros botones específicos de la aplicación o iconos.

  • Icono Acciones . Contiene funciones específicas de la aplicación para organizar objetos.

  • Botón Vista. Contiene funciones para controlar la disposición del área de visualización y para abrir o cerrar la vista de carpeta.

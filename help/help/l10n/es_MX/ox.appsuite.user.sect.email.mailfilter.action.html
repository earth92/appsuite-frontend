5.8.3. Añadir una acción

Use una acción para definir que sucede con un correo que coincide con las condiciones. Puede establecer una o varias acciones.

Cómo añadir una acción en la ventana Crear nueva regla:

  1. Presione en Añadir acción. Seleccione una acción del menú.

  2. Dependiendo de la acción, se mostrarán más botones y campos de datos. Use tales funciones para completar la acción.

  3. Se pueden añadir más acciones.

    Para borrar una acción, presione el icono Borrar que hay junto a la acción.

Acción de nivel superior:

  • Creación de nuevas reglas

  • Cifrado de correos entrantes mediante una regla de filtrado

6.6. Adición de listas de distribución

Aprenda a crear listas de distribución. Las listas de distribución se pueden usar para enviar correos, invitaciones a citas o elementos compartidos, a varias personas introduciendo el nombre de la lista de distribución en lugar de la dirección de correo de cada persona.

Existen las siguientes posibilidades:

  • Creación de nuevas listas de distribución

  • Almacenamiento de destinatarios de correo en forma de lista de distribución

  • Creación de una lista de distribución a partir de una lista de participantes

Interfaz de usuario:

  • La barra de herramientas de Libreta de direcciones

4.2.5. Creación de una cuenta de Xing

Si aún no dispone de una cuenta de Xing, puede usar un widget para crear una nueva cuenta de Xing.

Cómo crear una cuenta de Xing usando sus datos del software colaborativo:

  1. Presione en Añadir widget en la parte superior derecha de la aplicación Portal. Seleccione Xing. El widget se añadirá.

  2. Presione en Crear una cuenta de XING usando los datos aquí almacenados, en el widget.

  3. Compruebe los datos sugeridos para crear la cuenta de Xing. Para crearla, presione en Crear.

7.9.7. 予定をインポート

iCal形式のファイルから予定をインポートできます。

iCalファイルから予定をインポートする：

  1. 予定表を選択して、フォルダービューに予定をインポートします。

  2. 予定表の隣のアクションアイコンをクリックする。インポート をクリックします。

  3. ファイルからインポートウィンドウでファイルをアップロードをクリックします。iCal フォーマットのファイルを選択します。

    既存の予定と同じidの予定をインポートするには、既存の予定を無視を有効にします。

  4. インポート をクリックします。

結果：連絡先が予定表に追加されます。

注：インポートされた予定の参加者全員が削除されます。代わりに、あなたが参加者として追加されます。

8.6. 仕事の編集

タスクのデータを後で編集する方法を説明します。

前提条件: 仕事が保存されているフォルダーにオブジェクトを編集する権限が必要です。

仕事を編集する：

  1. 一覧から仕事を選択します。

  2. ツールバーで、編集 をクリックします。仕事のデータが表示されます。

  3. データを編集します。

  4. 保存をクリックします。

メッセージの操作

10.3.1. メッセージの読み込み、送信、返信

次のようなオプションの機能があります。

  • チャットのメッセージを検索し読み込む受信メッセージの通知方法：

    • 未読のメッセージがある場合、メニューバーのチャットアイコンには赤い点の印がつきます。

    • チャット設定で次の通知が設定可能です。

      • 受信メッセージの通知を鳴らす。

      • グループウェアがバックグラウンドにある場合、デスクトップ通知が表示される。

      • オフラインの場合、チャットメッセージの内容を含んだメール通知を受信します。このメールに返信すると、返信がチャットに挿入されます。

  • グループまたは一人の連絡先にメッセージまたはファイルを送信

  • 以前のメッセージに返信します。元のメッセージが返信に引用されます。

メッセージを読む：

  1. チャットリストからチャットを選択します。

    未読メッセージとのチャットは、次のように表示されます：

    • チャット名は太字で表示されます。

    • 数字は未読メッセージの数です。

  2. スレッドのメッセージを読みます。そのスレッドの前のメッセージを見るには、右罫線のアイコンを使って上にスクロールします。

オプション：

  • チャットを検索するには、新規チャットを検索または開始入力フィールドに検索条件を入力します。

メッセージに送信する：

  1. 次のいずれかの方法を使用します。

    • 新規のチャットを開始するには、新規アイコンをクリックします。

      メニューでスレッドタイプを選択します。

    • 既存のチャットで行う場合、チャットリストからチャットを選択します。いままでやり取りされたチャットのメッセージが表示されます。

  2. 入力フィールドにメッセージを入力します。

    メッセージの入力中にチャットウィンドウを閉じる、または別のチャットを開いた場合、入力した内容はグループウェアにサインインしている間は維持されます。

  3. メッセージを送信するには、 [Enter] を押すか、 送信 アイコン をクリックします。

    プライベートでは、メッセージ横のアイコンがメッセージの既読状態を示します。

    • グレーチェック印1個：メッセージがサーバーに届いています。

    • グレーチェック印2個：メッセージがサーバーによって受取人に転送されました。

    • ブルーチェック印2個：メッセージが受取人によって既読になりました。

オプション：

  • メッセージ内で改行するには、[Shift]+[Enter]を押してください。

  • グループにメッセージを送る場合、具体的に連絡先を指定することができます。指定するには、[@]の後に連絡先の名前とメッセージを入力します。指定された連絡先は、チャットリストのグループ名に赤い印で示されます。

  • 絵文字を送信するには、入力フィールド横の絵文字を追加アイコンをクリックします。絵文字のウィンドウが開きます。少なくとも1つの絵文字を選択してください。

    ウィンドウを閉じるには、もう一度絵文字を追加アイコンをクリックします。

  • ファイルを送信するには、入力フィールド横のファイルをアップロードアイコンをクリックします。

    ファイル ブラウザやデスクトップから一つまたは複数のドキュメントをドラッグしてチャットフィールド上にドロップしてもファイルを添付できます。

    注：安全のために、実行可能なファイルは送信できません。

  • 以前に送信された、または受信されたファイルすべてを閲覧するには、チャットリストのすべてのファイルをクリックします。

  • メッセージ本文の書式設定をするには、次のキャラクターを使用します。

    • この言葉は_イタリック体_で印刷されます。

    • この言葉は*太字*で印刷されます。

    • この言葉は~斜線~で印刷されます。

    • これはライン内の`プログラムコードです。

    • これはブロックの```プログラムコードです。

      コードライン1

      コードライン2

      ```

      注：プログラムコード内で改行するには、 [Shift]+[Return]を使用してください。

メッセージに返信する：

  1. チャットフィールドで、メッセージに行きます。

    メッセージの隣のメッセージアクションアイコンをクリックします。返信を選択します。

    メッセージとその送信者がメッセージフィールドに引用されます。

  2. メッセージフィールドに返信を入力します。

    返信を送信するには、 [Enter] を押すか、 送信 アイコン をクリックします。

10.2.2. Modificare le chat

Le chat di gruppo e i canali pubblici possono essere modificati da tutti i partecipanti. Sono disponibili le seguenti opzioni:

  • cambiare l'immagine

  • cambiare il nome

  • Per le chat di gruppo: aggiungere o eliminare partecipanti

Come modificare una chat di gruppo o un canale pubblico:

  1. Selezionare una chat di gruppo o un canale pubblico dall'elenco delle chat.

  2. Fare clic sull'icona Altre azioni in alto a destra.

    Selezionare Modifica chat o Modifica canale. Si aprirà una finestra di modifica.

  3. È possibile modificare quanto segue:

    • cambiare l'immagine facendo clic su di essa

    • cambiare il nome

    • Per le chat di gruppo: aggiungere o eliminare partecipanti.

  4. Fare clic su Salva.

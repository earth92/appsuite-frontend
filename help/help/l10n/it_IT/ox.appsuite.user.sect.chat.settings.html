10.5. Impostazioni di chat

Come utilizzare le impostazioni delle chat:

  1. Fare clic sull'icona Impostazioni a destra della barra dei menu. Fare clic su Impostazioni.

  2. Fare clic su Chat nell'elenco.

In base alla configurazione, sono disponibili le seguenti impostazioni:

Opzioni di visualizzazione

  • selezionare l'ultima chat all'avvio. Definisce se la chat utilizzata in precedenza sarà visualizzata nella finestra della chat dopo l'accesso al groupware.

  • Raggruppare le chat per tipo. Definisce se le chat saranno raggruppate in chat private, chat di gruppo e canali.

  • Ordina chat per. Definisce come saranno ordinate le chat nell'elenco delle chat.

  • densità dell'elenco. Definisce il livello di dettaglio della visualizzazione delle chat nell'elenco delle chat.

Notifiche

  • mostrare le notifiche del desktop per i nuovi messaggi. Definisce se si riceverà una notifica del desktop per i messaggi in arrivo mentre il groupware è in esecuzione sullo sfondo. Questa impostazione avrà effetto solo se sono state abilitate anche le notifiche del desktop nelle impostazioni di base, vedere Lavorare con le notifiche

  • riprodurre un suono di notifica per i nuovi messaggi. Definisce se sarà riprodotto un suono per i messaggi in arrivo. Se i suoni di notifica sono abilitati, sono disponibili impostazioni aggiuntive:

    • il suono da riprodurre

    • le condizioni per la riproduzione di un suono

  • notifiche dei messaggi di posta non in linea. Definisce se e per quali chat si riceverà una notifica tramite posta elettronica se si è disconnessi.

10.2.5. Chiudere una chat

Per mantenere una panoramica chiara dell'elenco delle chat, è possibile chiudere o riaprire le chat.

Come chiudere una chat:

  1. Selezionare una chat privata o di gruppo o un canale pubblico dall'elenco delle chat.

  2. Fare clic sull'icona Altre azioni in alto a destra.

    Selezionare Chiudi chat.

La chat nell'elenco e i messaggi nel campo della chat saranno chiusi. Icone per la creazione di una nuova chat saranno visualizzate nel campo della chat.

Come aprire una chat chiusa:

  1. Fare clic su Cronologia nell'elenco delle chat. Tutte le chat saranno visualizzate nel campo della chat.

  2. Fare clic su Apri accanto a una chat.

10.2.6. Abbandonare una chat

È possibile lasciare una chat di gruppo o un canale pubblico in qualsiasi momento. Dopodiché non sarà più leggere i messaggi della chat abbandonata. Nota: a differenza dei canali pubblici, non è possibile partecipare di nuovo a una chat di gruppo di propria iniziativa.

Come abbandonare una chat di gruppo o un canale pubblico:

  1. Selezionare una chat di gruppo o un canale pubblico dall'elenco delle chat.

  2. Fare clic sull'icona Altre azioni in alto a destra.

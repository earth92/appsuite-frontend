8.9.8. Eliminare le attività

È possibile eliminare le attività.

Avviso: se si elimina un'attività, non sarà più possibile recuperarla.

Come eliminare le attività:

  1. Selezionare una o più attività dall'elenco.

  2. Fare clic su Elimina nella barra degli strumenti.

  3. Confermare che si desidera eliminare l'attività.

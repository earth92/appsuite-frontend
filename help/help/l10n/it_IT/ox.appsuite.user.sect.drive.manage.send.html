9.7.1. Inviare file come allegati di posta

È possibile inviare le versioni correnti dei file come allegati di posta.

Come inviare file come allegati di posta elettronica:

  1. Selezionare uno o più file nell'area di visualizzazione.

  2. Utilizzare uno dei seguenti metodi:

    • Fare clic sull'icona Altre azioni nella barra degli strumenti. Fare clic su Invia tramite posta.

    • Fare clic su Invia tramite posta nel menu contestuale.

    • Utilizzare l'icona Altre azioni nel visualizzatore.

  3. Nella finestra di modifica del messaggio di posta, compilare i dettagli per inviare un nuovo messaggio.

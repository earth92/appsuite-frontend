9.7.5. Copiare i file

È possibile copiare file in un'altra cartella: se si copia un file condiviso, i permessi non saranno copiati. Ciò significa che la copia del file non è condivisa.

Prerequisito: si dispone dei permessi appropriati per creare oggetti nella cartella di destinazione.

Come copiare file in un'altra cartella:

  1. Selezionare uno o più file nell'area di visualizzazione.

  2. Utilizzare uno dei seguenti metodi:

    • Fare clic sull'icona Altre azioni nella barra degli strumenti. Selezionare Copia.

    • Selezionare Copia dal menu contestuale.

  3. Selezionare una cartella nella finestra Copia. È possibile creare una nuova cartella facendo clic su Crea cartella.

    Fare clic su Copia.

10.4. Creare chat in altre applicazioni

È possibile creare una chat mentre si lavora con gli oggetti del groupware in altre applicazioni.

Esistono le seguenti opzioni:

  • Risposta a un messaggio di posta elettronica con un messaggio di chat

  • Iniziare una chat con i contatti

  • Iniziare una chat con i partecipanti all'appuntamento

Interfaccia utente:

  • La finestra di Chat

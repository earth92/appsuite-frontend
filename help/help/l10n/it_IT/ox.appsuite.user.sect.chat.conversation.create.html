10.2.1. Creare chat

Esistono le seguenti opzioni:

  • creare una chat privata con un singolo contatto

  • creare una chat di gruppo con partecipanti specifici

  • creare un canale pubblico a cui possono partecipare altri contatti

Come creare una chat:

  1. Fare clic sull'icona Nuovo accanto all'elenco delle chat o su Nuova chat.

    Si aprirà un menu.

  2. Utilizzare uno dei seguenti metodi:

    • Per creare una chat con un singolo contatto, selezionare Chat privata. Si aprirà la finestra Avvia nuova conversazione.

      Selezionare un contatto.

      Fare clic su Avvia conversazione.

    • Per creare una chat con più partecipanti, selezionare Chat di gruppo. Si aprirà la finestra Crea chat di gruppo.

      Digitare un nome per il gruppo. È possibile aggiungere un'immagine facendo clic sull'icona.

      Aggiungere contatti.

      Fare clic su Crea chat.

    • Per creare un canale pubblico a cui possono partecipare altri contatti, selezionare Canale. Si aprirà la finestra Crea nuovo canale.

      Digitare un nome per il canale. È possibile aggiungere un'immagine facendo clic sull'icona.

      Fare clic su Crea canale.

La nuova chat sarà visualizzata nell'elenco.

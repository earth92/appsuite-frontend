4.1.6. Il widget Attività

Mostra le attività non completate. È possibile effettuare le seguenti operazioni:

  • Facendo clic su un'attività, si apre una finestra a comparsa. La finestra a comparsa mostra i dati dell'attività.

  • Per modificare l'attività, fare clic sul pulsante relativo nella finestra a comparsa.

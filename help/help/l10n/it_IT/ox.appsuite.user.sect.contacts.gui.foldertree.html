6.1.2. La vista delle cartelle di Rubrica

Visualizza le rubriche.

Per visualizzare la vista delle cartelle, fare clic su Vista nella barra degli strumenti. Abilitare Vista delle cartelle.

Contenuto

  • Le mie rubriche. Contiene le proprie rubriche personali.

  • Aggiungi nuova rubrica. Apre un menu con le funzioni per la creazione di rubriche e per la sottoscrizione di rubriche esterne o condivise.

  • I miei dati di contatto. Apre una finestra per cambiare i dati di contatto nella rubrica globale.

  • Rubriche pubbliche. Contiene rubriche condivise con tutti gli utenti.

  • Rubriche condivise. Contiene rubriche condivise con voi da altri utenti.

In base alla configurazione, le rubriche pubbliche e le rubriche condivise potrebbero non essere disponibili.

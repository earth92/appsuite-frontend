Lavorare con i messaggi

10.3.1. Leggere, inviare, rispondere ai messaggi

Esistono le seguenti opzioni:

  • Cercare e leggere i messaggi di una chat. Modalità di notifica dei messaggi in arrivo:

    • Se sono presenti messaggi non letti, l'icona della chat nella barra dei menu sarà contrassegnata da un punto rosso.

    • È possibile impostare le seguenti notifiche nelle impostazioni della chat:

      • Sarà riprodotto un suono per i messaggi in arrivo.

      • Se il groupware è in esecuzione sullo sfondo, sarà visualizzata una notifica sul desktop.

      • Se non si è in linea, si riceverà una notifica tramite posta elettronica con il contenuto del messaggio di chat. Se si risponde a questo messaggio, la risposta sarà inserita nella chat.

  • inviare messaggi e file a un gruppo o un singolo contatto

  • Rispondere a un messaggio precedente. Il messaggio originale sarà citato nella propria risposta.

Come leggere un messaggio:

  1. Selezionare una chat dall'elenco delle chat.

    Le chat con messaggi non letti saranno visualizzate come segue:

    • Il nome della chat sarà visualizzato in grassetto.

    • Una cifra indica il numero di messaggi non letti.

  2. Leggere i messaggi della conversazione. Per visualizzare i messaggi precedenti in questa conversazione, utilizzare le icone sul bordo destro per scorrere verso l'alto.

Opzioni:

  • Per cercare le chat, inserire un termine di ricerca nel campo di immissione Cerca o avvia una nuova chat.

Come inviare un messaggio:

  1. Utilizzare uno dei seguenti metodi:

    • Per iniziare una nuova chat, fare clic sull'icona Nuovo .

      Selezionare un tipo di conversazione nel menu.

    • Per continuare con una chat esistente, selezionare una chat dall'elenco. Saranno visualizzati i messaggi della chat scambiati fino a quel momento.

  2. Digitare un messaggio nel campo di immissione.

    Se si chiude la finestra della chat o si apre un'altra chat mentre si digita il proprio messaggio, quanto digitato sarà mantenuto finché si rimane connessi al groupware.

  3. Per inviare il messaggio, premere [Invio] o fare clic sull'icona Invia .

    Per le chat private, un'icona accanto al messaggio indica lo stato di lettura del messaggio:

    • Un segno di spunta grigio: il messaggio è arrivato sul server.

    • Due segni di spunta grigi: il messaggio è stato trasferito dal server al destinatario.

    • Due segni di spunta blu: il messaggio è stato letto dal destinatario.

Opzioni:

  • Per creare un'interruzione di riga all'interno di un messaggio, premere [Maiusc]+[Invio].

  • Se si invia il messaggio a un gruppo, è possibile indirizzarlo a un contatto in modo specifico. Per fare ciò, digitare [@] seguito dal nome della persona e dal messaggio. Il contatto indirizzato vedrà un contrassegno rosso in corrispondenza del nome del gruppo nell'elenco delle chat.

  • Per inviare emoji, fare clic sull'icona Aggiungi emoji accanto al campo di inserimento. Si aprirà una finestra con gli emoji. Selezionare almeno un emoji.

    Per chiudere la finestra, fare clic sull'icona Aggiungi emoji .

  • Per inviare file, fare clic sull'icona Carica file accanto al campo di inserimento.

    È possibile anche trascinare uno o più file da un navigatore di file o dal desktop al campo della chat.

    Nota: per motivi di sicurezza, non è possibile inviare file eseguibili.

  • Per visualizzare tutti i file precedentemente inviati o ricevuti, fare clic su Tutti i file nell'elenco delle chat.

  • Per formattare il testo di un messaggio, utilizzare i seguenti caratteri:

    • Questa parola è stampata in _corsivo_.

    • Questa parola è stampata in *grassetto*.

    • Questa parola è ~barrata~.

    • Questo è il `codice di programma` all'interno di una riga.

    • Questo è ```codice di programma in un blocco

      Riga di codice 1

      Riga di codice 2

      ```

      Nota: per le interruzioni di riga nel codice del programma, utilizzare [Maiusc]+[Invio].

Come rispondere a un messaggio:

  1. Nel campo della chat, portarsi su un messaggio.

    Fare clic sull'icona Azioni dei messaggi accanto al messaggio. Selezionare Rispondi.

    Il messaggio e il suo mittente saranno citati nel campo del messaggio.

  2. Digitare la propria risposta nel campo del messaggio.

    Per inviare la risposta, premere [Invio] o fare clic sull'icona Invia .

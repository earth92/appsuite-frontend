8.9.5. Görevleri içe aktarma

iCal biçimindeki dosyalardan görevleri içe aktarabilirsiniz.

iCal dosyalarındaki görevleri içe aktarma:

  1. Klasör görünümünde, görevlerin içeriği alınacağı klasörü seçin.

  2. Görev klasörünün yanındaki Eylemler simgesine tıklayın. İçe aktar'a tıklayın.

  3. Dosyadan içe aktar penceresinde Dosyayı yükle'ye tıklayın. iCal biçiminde bir dosya seçin.

    Mevcut görevlerle aynı ID'ye sahip görevleri de aktarmak isterseniz Mevcut etkinlikleri görmezden gel'i işaretleyin.

  4. İçe aktar'a tıklayın.

Sonuç: Görevler klasöre eklenir.

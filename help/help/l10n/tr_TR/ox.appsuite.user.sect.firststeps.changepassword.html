2.3. Parola değiştirme

Not: Yapılandırmanıza bağlı olarak, parola değiştirme adımları burada gösterilenden farklı olabilir. Bu durumda sistem yöneticinize veya hosting sağlayıcınıza başvurabilirsiniz.

Parolanızı değiştirme:

  1. Menü çubuğundaki Tüm uygulamalar simgesine tıklayın. Uygulama başlatıcıda Portal öğesine tıklayın.

  2. Kullanıcı verileri widget'ı görünmüyorsa sağ üstteki Widget ekle düğmesine tıklayın. Kullanıcı verileri'ne tıklayın.

    Kullanıcı verileri widget'ındaki Parolam'a tıklayın.

  3. Parolanızı değiştirin.

Kullanıcı arayüzü:

  • Kullanıcı bilgileri widget'ı
